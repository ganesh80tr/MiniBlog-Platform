# MiniBlog-Platform
MiniBlog is a RESTful blog application developed using Node.js, Express, and EJS. The project implements full CRUD functionality, uses UUIDs for unique post identification, and method-override to support PATCH and DELETE requests.

# Project Overview
MiniBlog Platform is a lightweight blogging application built using Node.js, Express, and EJS.
It focuses on core blogging concepts like CRUD operations, RESTful routing, and server-side rendering,
without using a database. Data is managed in-memory using arrays, making it perfect for learning
backend fundamentals.

# Key Features
- Create, Read, Update, and Delete blog posts (CRUD)
- RESTful API structure using Express
- Dynamic views rendered with EJS templates
- Unique post IDs using UUID
- Method override for PATCH and DELETE requests
- Clean and minimal UI
- No database used (in-memory data handling)
